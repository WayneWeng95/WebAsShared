# Peak Memory Footprint (incl. external KV storage) — per workload & load

All values are MB. `total = peak_mem_mb + KV-storage`; WasMem has no
external KV (shared-memory substrate). Baseline KV storage = resident
bytes written (`kvs_ser_mb`/`kvs_put_mb`/`state_kv_mb`); read traffic
(`kvs_get_mb`) is excluded. CRASH = OOM in our sweep.

**Faasm** `peak_mem_mb`: **WordCount** is the high-water Σ RSS of the whole
driver process tree (driver + co-resident Faaslets), the same whole-tree
accounting WasMem applies to its host tree — measured the same way on both
sides. The remaining workloads still report the older per-Faaslet concurrent
footprint (Σ private RSS + shared-once) pending the same migration.
